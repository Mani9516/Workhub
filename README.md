## 🏢 WorkHub – Employee Self-Service Portal

WorkHub is a modern Employee Self-Service (ESS) platform developed using **React, FastAPI, SQLite, and Docker** to streamline HR operations and improve employee engagement. It provides a centralized digital workspace where employees, managers, and HR teams can access essential services through secure role-based dashboards.

The platform includes core modules such as **Attendance Management, Leave Management, Payroll, Learning & Development, Career Growth, Wellness, Compliance, and Employee Profiles**. Employees can apply for leave, track attendance, view payslips, access company documents, and monitor learning progress from a single interface. Managers can approve requests and monitor team activities, while HR administrators can manage employee records, policies, and organizational settings.

WorkHub uses **JWT-based authentication and role-based access control (RBAC)** to ensure secure and personalized user experiences. The responsive Material-UI design provides seamless access across desktop, tablet, and mobile devices.

A key innovation of the platform is its **AI-powered recommendation engine**, which analyzes employee skills, interests, department, certifications, and career goals to suggest relevant training programs and compliance policies. This helps employees continuously improve their skills and supports career development within the organization.

The platform also features **Echo**, an intelligent AI chatbot that assists users with leave policies, compliance information, learning recommendations, and portal navigation. By providing instant responses and personalized guidance, Echo reduces HR workload and enhances user satisfaction.

WorkHub follows clean architecture principles with a well-structured frontend and backend, secure API communication, proper error handling, and Docker-based deployment. The application demonstrates modern software engineering practices while delivering a complete, scalable, and user-friendly HR management solution.

Overall, WorkHub transforms traditional HR processes into an intelligent, self-service experience that improves productivity, transparency, compliance, and employee engagement across the organization.
